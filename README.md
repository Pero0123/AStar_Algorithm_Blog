# A* Algorithm - C++ Project Blog

My name is Pero Schwitalla and this is my project blog documenting the development of an A* pathfinding algorithm in modern C++, created as part of my Bachelor of Engineering (Honours) in Software and Electronic Engineering.

## About

I built an A* pathfinding algorithm in C++, following Google's C++ Style Guide throughout. The blog covers my full development process including:
- How the A* algorithm works and my research into it
- The heuristics I explored: Euclidean, Manhattan, and Diagonal distance
- Grid generation using random values, Perlin noise, and Worley noise via FastNoiseLite
- My class design and core algorithm logic
- How I tested the application
